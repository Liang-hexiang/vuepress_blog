# Sublime Text使用文档说明书

### SublimeText预览Markdown

#### 1. MarkdownEditor
:label:MarkdownEditing是一个功能丰富的插件，提供了对Markdown文件的语法高亮、自动补全、预览等功能。

安装：

Ctrl+Shift+P打开控制台，输入Package Control: Install Package，然后输入MarkdownEditor，选择安装
